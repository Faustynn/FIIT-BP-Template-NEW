## Download Tectonic

### 1) macOS

``` bash
brew install tectonic
```

### 2) Windows

``` bash
scoop install tectonic
```

GitHub Guide: https://github.com/tectonic-typesetting/tectonic

## Usage

``` bash
tectonic template.tex
```

P.S. Tectonic automatically handles `.bib` files and references, so you don't need to worry about them.
